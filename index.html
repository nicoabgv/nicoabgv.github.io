Kobra

Plataforma financiera que sostiene el día a día de Cybersur.

Kobra es la solución a medida que construí para Cybersur. Con un stack de FastAPI, Celery, MySQL y Docker, centraliza facturación, cobros y notificaciones para que el equipo financiero trabaje con información coherente y actualizada.

Los cronjobs programados gestionan exportaciones y limpiezas de datos, mientras que los pipelines de avisos y recordatorios dejan registro de cada acción. Todo el flujo está pensado para dar visibilidad inmediata y reducir tareas manuales repetitivas.

  • Workflows automáticos que reducen drásticamente los tiempos de seguimiento y cobranza.
  • Cronjobs trazables para exportaciones y rutinas de limpieza que mantienen los datos consistentes.
  • Paneles internos con métricas esenciales para el seguimiento de cobros y pendientes.

Siguiente iteración: ampliar reportes financieros internos y automatizar conciliaciones.

Diagrama técnico 
Equipo Cybersur
    ↓
FastAPI API → MySQL
    │
    ├─ Celery workers → Emails y recordatorios
    └─ Cronjobs programados → Reportes CSV

BeatList

Gestión profesional de playlists con sello oficial de Spotify.

BeatList es la plataforma para melómanos, curadores y creadores que necesitan algo más potente que el gestor nativo de Spotify. Reúne administración avanzada, descubrimiento y analítica en una experiencia fluida.

Gracias a la integración oficial con la API de Spotify y a la autenticación OAuth 2.0 aprobada, el backend en Flask accede a datos extendidos con estabilidad profesional. El frontend en JavaScript ofrece dashboards y un reproductor web con microinteracciones personalizadas.

  • Fusión, depuración y reorganización de playlists con reglas inteligentes basadas en métricas como energía o bailabilidad.
  • Motor de recomendaciones por artistas y géneros favoritos, sumado a dashboards que muestran hábitos de escucha.
  • Persistencia en MySQL para conservar estadísticas personalizadas y ofrecer continuidad entre sesiones.

Posibles evoluciones: sumar modo colaborativo o exportaciones de insights en futuras versiones.

Diagrama técnico 
Cliente web
    ↓
Flask API (OAuth 2.0 Spotify)
    ↓
MySQL → Historial y métricas
    ↓
Dashboards JS + Reproductor

MesasYa

Reservas inteligentes para cadenas de restauración.

MesasYa es una plataforma SaaS en producción que digitaliza la gestión de reservas para restaurantes con múltiples sedes. Ofrece control granular de ocupación, automatización de comunicaciones y dashboards listos para el gestor.

Con Flask, Celery, MySQL y Docker como base, la solución se despliega con Docker Compose y buenas prácticas de seguridad, manteniendo workflows personalizables por zona, servicio y equipo. Las analíticas y exportaciones se adaptan a las decisiones del negocio.

  • Planos de sala interactivos y configurables por sede, zona y servicio.
  • Notificaciones automáticas para equipos internos y clientes finales.
  • Indicadores de demanda, capacidad y rendimiento exportables para decisiones estratégicas.

Próximos pasos: integrar informes automáticos y seguir mejorando la experiencia del gestor.

Diagrama técnico 
Staff y reservas online
    ↓
Flask API → MySQL
    │
    ├─ Celery workers → Emails & SMS
    └─ Docker Compose → Servicios coordinados
