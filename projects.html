BeatList

BeatList es una plataforma innovadora para gestionar y analizar playlists de Spotify de manera intuitiva. Algunas características incluyen:

  • Ordenar y fusionar playlists personalizadas
  • Analizar estadísticas detalladas de las canciones
  • Generar listas personalizadas basadas en gustos
  • Reproductor de música integrado para una experiencia fluida

Desarrollado con Flask, JavaScript y la API de Spotify, BeatList es tu aliado para optimizar la experiencia musical.

Spotify Analyzer

Spotify Analyzer es un programa en Python que analiza todo tu historial de escucha de Spotify, almacenado en archivos JSON. Proporciona estadísticas detalladas y visualizaciones basadas en tus datos de escucha. Algunas características incluyen:

  • Muestra el número total de canciones escuchadas
  • Tiempo total dedicado a escuchar música
  • Muestra las 5 canciones y artistas más escuchados
  • Estadísticas anuales y patrones de escucha diaria
  • Exportación de estadísticas a Excel

Desarrollado con Python, matplotlib, y tkinter, Spotify Analyzer permite visualizar y exportar tus estadísticas de escucha.
